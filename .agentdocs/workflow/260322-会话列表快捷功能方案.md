# 260322 会话列表快捷功能方案

## Task Overview

用户要求继续增强会话列表的快捷功能，目标是让左侧 sidebar 会话列表本身支持更直接的快捷操作，而不只是点击打开。

本轮按最小闭环落地：

1. sidebar 会话项支持 **重命名**
2. sidebar 会话项支持 **导出**
3. sidebar 会话项支持 **删除**

## Complexity Assessment

- Atomic steps: 3–4 → 0
- Parallel streams: no → 0
- Modules/systems/services: web Layout + session transfer + tests → +1
- Long step (>5 min): no → 0
- Persisted review artifacts: yes → +1
- OpenCode available: yes → -1
- **Total score**: 1
- **Chosen mode**: Lightweight
- **Routing rationale**: 这是一个中等规模的 sidebar 交互增强，主要改 Layout 并复用 SessionsPage 的现有动作模式，不需要额外 runtime 计划目录。

- 不破坏原有点击打开会话行为
